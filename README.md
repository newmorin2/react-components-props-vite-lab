# Underreacted Blog

A personal blog site built with React, using components and props.

## How to run
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
4. Open your browser at `http://localhost:5173`

## Components
- **App** - Root component, imports blog data and passes it to child components.
- **Header** - Displays the blog name
- **About** - Displays the blog logo and description.
- **ArticleList** - Displays all blog posts
- **Article** - Displays a single blog post
